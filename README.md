# gqThreadedMapLimit

A module for running map limit jobs using worker threads with better performance

## Getting Started

### Dependencies

  * dependencies: "async": "3.2.6"
  *
  * devDependencies:
  * "coffeescript": "2.7.0",
  * "gqtest": ">=0.0.17",
  * "nodemon": "3.1.10"

### Installing

* npm run-script install:nobin

### Executing program

* gqThreadedMapLimit=require("gqThreadedMapLimit")
